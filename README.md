# MathsQuestionsAPI (Work In Progress)

## Description

A free & easy-to-use API for obtaining mathematical-related questions, categorised by topics such as algebra and arithmetic.

## Data Structure

```
{
    "arithmetic": {
        "q1": {
            "question": "What is 100/25?",
            "options": ["3", "4", "5", "6", "7"],
            "correctIndex": 1
        }
    },
    "algebra": {
        "q1": {
            "question": "What is 100/25?",
            "options": ["3", "4", "5", "6", "7"],
            "correctIndex": 1
        }
    }
    ...
}
```
